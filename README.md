# Topics in Algorithmic Data Analysis
This repository contains submitted reports for assignments from the course Topics in Algorithmic Data Analysis at the University of Saarland.

Every assignment firstly presents a reading task with multiple ideas from the research field for a particular topic, sometimes the ideas manage to create scientific discourse, which is where each assignment is trying to lend. 

Every such report requires decent understanding of the particular topic, as well as incorporating critical thinking to comprehend different aspects of each methodology.
